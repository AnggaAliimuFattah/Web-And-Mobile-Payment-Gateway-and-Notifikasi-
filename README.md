# Web-And-Mobile-Payment-Gateway-and-Notifikasi-

### Deskripsi 

Sebuah Aplikasi Monnitoring dan Pembayaran SPP Berbasis web dan mobille untuk siswa baru, yang dimana web berfungsi sebagai admin sekolah untuk mengetahui Informasi informasi Pembayaran yang sudah lunas dan belum lunas, serta mengingatkan dengan mengirim notifikasi yang meneruskan pesan ke WA, Mobile Apps berungsi sebagai Platform akses informasi Pembayaran dan Melakukan Pembayaran
